# python_example_repo
This repo is devoted to CS-25-3 students

Here is an example of code commands in terminal:

git init - initializes an empty repo in current folder

git remote add origin - binds a remote repo to your local repo

git add . - adds all files to current commit

git commit -m "first commit" - commits the changes with message "first commit"

git push origin master

git status - checks the status of modifications of your local repo

Other commands:

git branch - to know at which branch you are currently in

git checkout -b branch_name - creates new branch in your local repo with name "branch_name"

git checkout branch_name - switches between branches (from current branch to branch_name)

git merge branch_name - merges the changes to your current branch from branch_name
